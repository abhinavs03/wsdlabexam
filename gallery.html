Online Tyre Alloy Store

This is an e-commerce website exclusively for tyre alloy wheels. The site allows users to browse through a variety of alloy wheels, check availability, and place orders online. It is built using HTML, CSS for the front-end and PHP, JavaScript for the back-end.

Online Spa Booking System

This platform allows users to book spa treatments according to their preferred date and time. It provides a user-friendly interface to select services, choose therapists, and schedule appointments. The project uses HTML, CSS for the front-end and Django, MySQL for the back-end.
